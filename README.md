# 🦖 Dino Runner

A fun Chrome Dino-inspired endless runner game built with React and Vite.

Unlike Chrome's Dino game, this one works even when your internet does. 🚀

---

## 🎮 Features

- Smooth jumping mechanics
- Endless obstacle generation
- Dynamic game speed
- High score tracking with Local Storage
- Lives system
- Bonus life every 10 points
- Day / Night mode
- Sound effects
- Pause / Resume support
- Space key controls
- Responsive game UI

---

## 🕹️ Controls

| Action         | Key   |
| -------------- | ----- |
| Jump           | Space |
| Start Game     | Space |
| Restart Game   | Space |
| Pause / Resume | P     |

---

## ❤️ Lives System

- Game starts with 3 lives.
- Earn +1 life every 10 points.
- Maximum lives: 6.

---

## 🏆 Scoring

- Each obstacle successfully passed increases your score.
- High scores are automatically saved in your browser.

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/dino-runner.git
```

Navigate to the project:

```bash
cd dino-runner
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
